# Nestor
Nestor, yaşlıların dijital dışlanma ve dolandırıcılık korkusunu kıran, Apple-esque sadelikte bir yapay zeka asistanıdır. Gemini API ile karmaşık ekranları analiz eder, işlemleri "torununa anlatır gibi" basitleştirir. En kritik farkı; şüpheli veya yüksek meblağlı işlemleri askıya alıp vasi onayına sunan güvenlik kalkanıdır.
